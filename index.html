Customer Churn Analysis Project

Uncovering Key Drivers Behind Customer Attrition

Project Overview

This telecom churn analysis project explores customer behavior, support interaction, subscription details, and financial history to understand the reasons behind churn. Insights were derived using SQL, Python (Pandas), and Power BI to empower strategic retention decisions.

Key Metrics

  • Total Customers: 1,000
  • Churned Customers: 242 (24.2%)
  • Non-Churned Customers: 758 (75.8%)
